## معرفی وب سرویس چابک اس ام اس
چابک اس ام اس یک وب سرویس کامل برای ارسال و دریافت پیامک و پیامک صوتی و مدیریت کامل خدمات دیگر است که براحتی میتوانید از آن استفاده کنید.

### عضویت و تست رایگان

اگر در وب سرویس چابک اس ام اس عضو نیستید میتوانید از [وب سایت چابک اس ام اس](https://chaboksms.ir) ثبت نام  و اکانت آزمایشی همراه با ۵۰۰ پیامک رایگان برای تست API دریافت نمایید.
